Python Introduction

What is Python?

Well, Already You Know, Python is a popular programming language. it was created by Guido van Rossum, and released in 1991.

It was used for:

  • Web Development
  • Software Development
  • Mathematics
  • System Scripting
  • And More.

What Can Python do?

Python Can be used on a server to create web applications.

Python can be used alongside software to create workflow.

Python can connect to database systems. It can also read and modify files.

Python can be used to handle big data and perform complex Mathematics.

Python can be used for rapid prototyping, or production-ready software development

Python Variables

Variables are containers for storing values.

One of the most powerful features of a programming language is the ability to manipulate variables.

A variable is a name that refers to a value. you want to assign the value.

The variables are important roles in programming. you can easy to create variables.

x = 10

y = "ten"

z = 10.0

Python Casting

If you want to specify the data type of variable, this can be done with casting.

x = str(3)

y = int(3)

z = float(3)
